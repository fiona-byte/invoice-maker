# Invoice maker

This is an invoice maker that lets one create invoices, with a wide range of templates to choose from

## Tech Stack

- Frontend: React (TypeScript)
- Styling: Tailwind CSS
- Deployment: Netlify

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm

### Installation

```js
  git clone https://github.com/fiona-byte/invoice-maker
  cd invoice-maker
  npm install
```
